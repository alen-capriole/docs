## Overview

The AI Market Briefing feature provides comprehensive market sentiment analysis powered by LangGraph workflows and machine learning models.

## Architecture

The system combines two components:

**Offline Knowledge Base**
Curated intelligence from Capriole newsletters, research publications, and media content, distilled using LLM summarization.

**Real-Time Analysis**
LangGraph workflow processing current indicator data to generate market insights across five analytical categories.
