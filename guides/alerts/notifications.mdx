## Email Notifications

Email alerts are enabled automatically for all users. Notifications are sent to your registered account email.

Add `noreply@capriole.com` to your email safe sender list to prevent alerts from going to spam.

## Discord Setup

1. Navigate to **Account Settings** → **Integrations** → **Discord**
2. Click **Connect Discord**
3. Authorize Capriole Charts bot
4. Choose notification method:
   - **Direct Messages**: Private alerts sent to DMs
   - **Server Channel**: Share alerts with your team

## Telegram Setup

1. Search for **@CaprioleChartsBot** in Telegram
2. Click **Start** or send `/start`
3. Get your connection code from **Account Settings** → **Integrations** → **Telegram**
4. Send the code to @CaprioleChartsBot
5. Confirmation message indicates successful connection

<Note>
Connection codes expire after 24 hours. Generate a new code if needed.
</Note>

## Testing Notifications

Test each connected channel from **Account Settings** → **Integrations**:
- Send test email
- Send test Discord message
- Send test Telegram message
