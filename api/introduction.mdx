## CAPI Backend Service

The Capriole Charts API (CAPI) provides programmatic access to financial market data through a RESTful interface. The service delivers real-time and historical data for all platform indicators.

## Technical Architecture

### Infrastructure

- **Framework**: FastAPI (Python)
- **Data Format**: Parquet-based storage for optimized query performance
- **Deployment**: Docker containers on AWS ECS
- **Storage**: AWS EFS for persistent data access
- **Caching**: Redis-based caching layer for improved response times

### Data Processing

The backend service handles:

- Parquet file reading and data transformation
- Time-series data aggregation
- Subscription-based access control
- Request caching and optimization
