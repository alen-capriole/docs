## Get started in three steps

### 1. Create an API key

Create your key from the Capriole AI API page.

Use the full secret key once. After creation, the UI shows only a masked preview.

### 2. Call `GET /v1/models`

Use this request to confirm your key works and to fetch the current public model list.

```python
import requests

API_KEY = "YOUR_API_KEY"
BASE_URL = "https://api.caprioletech.com"

response = requests.get(
    f"{BASE_URL}/v1/models",
    headers={"Authorization": f"Bearer {API_KEY}"},
    timeout=30,
)

response.raise_for_status()
print(response.json())
```

### 3. Call `POST /v1/chat/completions`

Send a simple text chat request.

```python
import requests

API_KEY = "YOUR_API_KEY"
BASE_URL = "https://api.caprioletech.com"

payload = {
    "model": "google/gemini-3.1-flash-lite-preview",
    "messages": [
        {
            "role": "user",
            "content": "Tell me a short joke."
        }
    ]
}

response = requests.post(
    f"{BASE_URL}/v1/chat/completions",
    headers={
        "Authorization": f"Bearer {API_KEY}",
        "Content-Type": "application/json",
    },
    json=payload,
    timeout=60,
)

response.raise_for_status()
print(response.json())
```

## Base URL

`https://api.caprioletech.com`
