## Why Capriole AI

Capriole AI gives you one API for leading AI models.

Keep one integration and switch between models as needed.

- One API for model access
- Switch between supported models without changing your integration
- Clear, simple pricing with token packs at much lower cost

## Configure once

Use `openai-latest`, `claude-latest`, or `google-latest` when you want Capriole AI to keep your API or coding agent on our recommended flagship model for that provider.

Use a concrete model ID when you need one exact model version for testing, audits, or reproducible runs.
